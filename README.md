# 🔑 Advanced Password Generation (Flask App)

یک اپلیکیشن تحت وب نوشته شده با **Python + Flask** برای تولید رمز عبورهای قوی و تصادفی.  
این برنامه به شما کمک می‌کند تنها با یک کلیک، پسوردهایی امن و غیرقابل حدس بسازید.

- **Python 3.10+**
- **Flask 2.3+**

### مراحل نصب
1. کلون کردن پروژه:
   ```bash
